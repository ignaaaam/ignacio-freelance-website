# ChatGPT Prompt For Visual Asset Generation

Use this when you want ChatGPT to generate concept visuals from real project facts without fabricating performance claims.

```text
You are helping me create concept visuals for my freelance developer website.

Rules:
- Do not invent client logos, brands, confidential screens, results, or metrics.
- If a real screenshot is missing, create a polished placeholder concept only.
- The visual style should feel premium, editorial, warm, credible, and modern.
- Avoid generic SaaS-purple aesthetics.
- Use a palette close to: warm ivory, deep espresso, terracotta accent, dark green accent.

I need:
1. A portfolio mockup concept
2. A social/OG image concept
3. Optional simple brand-mark ideas

Project context:
[PASTE PROJECT OR BRAND CONTEXT HERE]

Return:
- A short visual direction
- 3 image prompts for the portfolio mockup
- 2 image prompts for the OG/social image
- 3 minimal brand-mark concepts
- A note explaining which parts must later be replaced by real proof assets
```
