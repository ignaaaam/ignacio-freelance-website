On April 16, 2026, while Anthropic announced Claude Opus 4.7, OpenAI fired back with a massive **Codex for Mac** update that redefines what an AI coding agent can be. The competition between both platforms is no longer just about which model writes better code — it's about who controls more of your working environment.

## Codex now operates your Mac while you're in another app

The most striking new feature is **Computer Use**: Codex can operate in the background with its own cursor, clicking and typing in Mac applications while you continue working in other windows without any interference. The concept isn't new — Anthropic has been exploring this with Claude — but Codex's implementation makes it accessible to developers who don't want to manage complex APIs.

The practical implication: you can tell Codex "finish this feature while I'm in a meeting" and when you return, the work is done (or at least meaningfully advanced).

## Multiple parallel agents

Codex now lets you launch several simultaneous agents working in parallel on your Mac. Each can have its own context, tools, and objective. This opens the door to workflows where one agent writes tests while another implements the feature, or one refactors while another documents.

For small teams or freelance developers juggling multiple projects, this is a genuine productivity multiplier.

## Built-in browser with Atlas technology

Codex integrates an **in-app browser** powered by Atlas technology. The key differentiator: you can comment directly on web pages to give the agent precise instructions. Instead of describing "click the green button in the upper right corner of the registration form," you point directly at the element.

This closes the gap between designers/PMs and the agent — no more translating visual instructions into text descriptions.

## Image generation with GPT-image-1.5

Without leaving Codex, you can now generate images with **gpt-image-1.5**. For developers building prototypes or needing quick assets (icons, illustrations, mockups), eliminating the context switch between tools saves meaningful time.

## Scheduled tasks and memory between sessions

One of the more interesting announcements: Codex can **schedule future work for itself** and wake up automatically to continue a long-term task, potentially across days or weeks. This turns the agent into something closer to an asynchronous collaborator than a point-in-time tool.

Managing GitHub review comments, multiple terminal tabs, and SSH connection to remote devboxes also land in this update.

## 111 new plugins

OpenAI launches a curated collection of **111 Codex plugins** combining skills, app integrations, and MCP servers. The ecosystem grows with the same App Store logic: the more integrations, the harder it becomes to leave the platform.

## Codex vs Claude Code: who wins?

The honest answer is: it depends on your context. Claude Code remains the preferred option for pure coding projects with heavy accumulated context, while Codex bets hard on Mac desktop integration. If you work primarily on Mac and value visual environment control, Codex wins on UX. If you need the most powerful agent for complex engineering projects, Claude Code remains the benchmark.

What's undeniable: 2026 is the year AI coding agents stop being a curiosity and become a standard part of the professional development stack.
